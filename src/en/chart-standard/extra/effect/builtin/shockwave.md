# `shockwave`

Shockwave effect. Best used with a `progress` animation.

![Example](image/shockwave.png)

## Parameters

- `progress` (float, default `0.2`, range 0–1): Shockwave progress; 0 = start, 1 = end.
- `centerX` (float, default `0.5`, range 0–1): X coordinate of the shockwave center.
- `centerY` (float, default `0.5`, range 0–1): Y coordinate of the shockwave center.
- `width` (float, default `0.1`): Width of the shockwave.
- `distortion` (float, default `0.8`): Distortion amount.
- `expand` (float, default `10.0`): Expansion range.
