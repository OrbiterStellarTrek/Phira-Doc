# vignette

Vignette effect. Darkens or tints the screen edges; can mimic some Arcaea anomaly effects.

![Example](image/vignette.png)

## Parameters

- `color` (color, default black [r, g, b, a]): Edge color.
- `extend` (float, default `0.25`, range 0â€“1): How far the edge extends; larger = more darkened area.
- `radius` (float, default `15.0`): Size of the central bright area; smaller = larger affected area.
